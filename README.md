# UAS Praktikum Pemrograman Mobile - EndemikDB

Aplikasi Ensiklopedia Hewan dan Tumbuhan Endemik Indonesia. Aplikasi ini dibuat untuk memenuhi tugas UAS Praktikum Pemrograman Mobile.

## Fitur Utama
- **API Integration**: Mengambil data terbaru dari server secara online menggunakan Retrofit.
- **Local Database (Room)**: Menyimpan data secara offline di dalam HP dengan 2 tabel (Endemik & Favorit).
- **Kategori Fragment**: Pemisahan data otomatis antara kategori Hewan dan Tumbuhan.
- **Pencarian Real-time**: Mencari data hewan/tumbuhan berdasarkan nama.
- **Favorit**: Menyimpan data yang disukai ke daftar favorit tersendiri.
- **Dark & Light Mode**: Mendukung tema gelap dan terang sesuai pengaturan sistem.

## Teknologi yang Digunakan
- Bahasa: Java
- Library: Retrofit, Room Database, Glide (Image Loader), Material Design.
